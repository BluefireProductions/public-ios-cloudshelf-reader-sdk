Detailed Description

A BCLHighlight object represents a location within a book that renders as highlighted.

A highlight is initially created by a BCLReaderView instance. If a highlight's location represents a range, the reader view will render the related text as highlighted. If the location does not represent a range, there will be no visible highlight.

Creates a highlight.

A highlight is initially created by a BCLReaderView instance. An application may wish to save a reader view's highlights to local or remote storage. Then when a BCLReaderView instance is created at a later time, those highlights can be passed back in. This method exists to allow previously created highlights to be created again and passed to the reader view during initialization.
