Detailed Description

A BCLPageList object represents a list of pages for a given book.

Some EPUB3 documents define their own list of pages. In all other cases, a list of pages is automatically generated using an algorithm that breaks up each spine item into multiple parts. Page boundaries do not depend on screen size or the scale factor applied to text.

Returns the position of the given item within the page list.

This is a convenience method that translates the given item into a position within the page list. An application could use this information to set the position of a slider, for example.

Returns
A position in the range of 0.0 to 1.0 inclusive, or 0.0 if none is available.
