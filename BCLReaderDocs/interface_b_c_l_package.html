§ initWithBookPath:()

- (nullable instancetype) initWithBookPath: (nonnull NSURL *)  bookPath

Creates a package for a given book.

Generally there is no need to manually create instances of packages, but it can be useful in cases where a BCLReaderView is not needed, such as getting a book's cover image or title.

Parameters
bookPathThe local path of the book.
Returns
A BCLPackage object, or nil if there was a problem creating the package.
