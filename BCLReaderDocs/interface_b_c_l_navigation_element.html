- (NSArray<BCLPageListItem *>*) estimatedPageListItems
readnonatomicstrong

An approximation of the BCLPageListItem objects associated with this navigation element.

The relationship between navigation elements and page list items cannot always be reliably determined. Use this property only when an imperfect approximation is needed.
