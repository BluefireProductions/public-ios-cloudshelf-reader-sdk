Detailed Description

A BCLReaderSettings object represents the settings associated with a reader view.

Settings are used in conjunction with a BCLReaderView instance to affect the behavior and appearance of the reader view.

§ init()

- (nonnull instancetype) init

Creates a reader settings object with default values.

The values are undefined and may change in a future release. It is better to explicitly set all properties or to call the initializer that accepts all values.

Parameters
columnModeThe column mode.
isMediaOverlayTapToPlayEnabledWhether or not tap-to-play is enabled for media overlay.
isTextAlignmentJustifiedWhether or not to have the book justify text (align to the left and right). Enabling this suggests to the book that justification should be used, although some books may ignore this. Disabling this lets the book control text alignment.
marginAmountThe amount of blank space added to the left and right sides of the reader view. A value of 1.0 causes text to be rendered at its default size. Values less than 1.0 render smaller text, and values greater than 1.0 render larger text. The value must be between 0.1 and 10.0. Values outside this range are clamped.
textScaleA value that controls the size of text in a book. A value of 1.0 causes text to be rendered at its default size. Values less than 1.0 render smaller text, and values greater than 1.0 render larger text. The value must be between 0.1 and 10.0. Values outside this range are clamped.
themeThe theme.

§ marginAmount

- (double) marginAmount
readwritenonatomicassign

The amount of blank space added to the left and right sides of the reader view.

The value must be between 0.0 and 1.0 where 0.0 adds no margin and 1.0 adds the maximum allowed margin. Values outside this range are clamped.
