Detailed Description

A BCLSearchResult object represents a search result from a BCLSearchRequest.

Property Documentation

§ prefix

- (NSString*) prefix
readnonatomicstrong

The text to the left of the main search text.

For example, if searching for "nice day" and the book contains the phrase, "It's a nice day to be outside", the prefix would include "It's a ", the text would be "nice day", and the suffix would include " to be outside". The prefix and suffix provide context that can be useful to display in the application's user interface.
