Detailed Description

A BCLLocation object represents a location within a book.

A location can represent either a single location or a range that has a beginning and end.

Creates a location.

A location is initially created by a BCLReaderView instance. An application may wish to create a location manually for use with a BCLHighlight object. A location's CFI is considered to be a "partial" CFI because it doesn't include a reference to a spine item.
